# rocks

## Install

  1. Clone repo
     ```
     $ git clone https://github.com/maxmahlke/rocks.git
     $ [sudo] pip install -e .
     ```
  2. Install `rocks` `python` package
     ```
     $ cd rocks
     $ [sudo] pip install -e .
     ```
  3. Check out the documentation
     ```
     $ rocks docs
     ```
